### How to run project:
**firstly install packages**

```make install```

**then you should choose the game and run the command from the list:**

 * ```./bin/brain-games.js```
 * ```./bin/brain-calc.js```
 * ```./bin/brain-gcd.js```
 * ```./bin/brain-even.js```
 * ```./bin/brain-progression.js```
 * ```./bin/brain-prime.js```
